## Description

proudcat-rust - cat + :rainbow:

proudcat-rust is basically a variant of
[`proudcat`](https://github.com/obfusk/proudcat) /
[`pridecat`](https://github.com/lunasorcery/pridecat)
written in rust.

![screenshot](screenshot.png)

## Help

```bash
$ proudcat --help
```

## Installing

```bash
$ cargo install proudcat
```
